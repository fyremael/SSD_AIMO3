## Script entry points

- `scripts/aggregate_votes.py`: Aggregate extracted answers with majority vote
- `scripts/build_problem_manifests.py`: Normalize raw problem data into prompt/eval/metadata JSONL manifests
- `scripts/colab_hf_generate.py`: Generate JSONL completions with a Hugging Face causal LM on Colab GPU
- `scripts/colab_runtime_probe.py`: Probe Colab runtime capabilities and emit a JSON summary
- `scripts/colab_train_lora.py`: Run LoRA SFT on Colab GPU from a JSONL training dataset
- `scripts/common.py`: Shared config resolution, JSONL I/O, and utility helpers used across the repo.
- `scripts/compare_eval_runs.py`: Compare two eval runs and emit paired win/loss/flip reports
- `scripts/constraint_library.py`: Cheap auditable constraint checks and penalty scoring helpers for tropical reranking.
- `scripts/extract_answer.py`: Extract a canonical final integer answer from math traces
- `scripts/generate_self_samples.py`: Generate self-sampled math traces or synthesize dry-run equivalents
- `scripts/materialize_colab_bundle.py`: Materialize a Colab experiment config bundle from notebook parameters
- `scripts/mock_external_generator.py`: Mock external JSONL generator backend for smoke tests
- `scripts/mock_external_trainer.py`: Mock external training backend for smoke tests
- `scripts/run_eval_math.py`: Run math evaluation with optional tropical reranking backend
- `scripts/run_validation_ladder.py`: Run the A0/A1/A5 validation ladder and emit paired summaries
- `scripts/train_ssd_math.py`: Build an SSD math SFT dataset package from self-generated traces
- `scripts/tropical_rerank.py`: Apply tropical reranking to extracted math traces
- `scripts/update_docs.py`: Generate repo index, status, and summary documentation artifacts.
